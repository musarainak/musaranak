## Instalation

This project is based on jekyll ![Jekyll info](https://jekyllrb.com/docs/installation/)
~~~~
bundle install
sudo gem install jekyll bundler
npm install
bundle exec jekyll serve
gulp
~~~~
Open browser ![http://127.0.0.1:4000/](http://127.0.0.1:4000/)
